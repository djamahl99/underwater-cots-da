# Unsupervised Domain Adaptation for Underwater Object Detection
## Requirements
* python3.8.15
* pytorch==1.12.1
* mmyolo==0.1.3
## Datasets
**Kaggle**: Kaggle COTS dataset

**AIMS**: Aims COTS dataset
## Testing
AIMS
```
 python evaluate.py --datset aims
```
Kaggle
```
python evaluate.py --datset kaggle
```
